### Implementation Requirements

#### 1️⃣ Define Plan Order

---

#### 2️⃣ Disable Logic (Authoritative)

Only disable the **current paid plan**.

```ts
function isPlanDisabled(targetPlan: PlanTier, currentPlan: PlanTier) {
  return currentPlan !== 'free' && targetPlan === currentPlan
}
```

---

#### 3️⃣ Upgrade Detection

```ts
function isUpgrade(target: PlanTier, current: PlanTier) {
  return PLAN_ORDER[target] > PLAN_ORDER[current]
}
```

---

#### 4️⃣ Click Handling (Canonical)

```ts
function handlePlanClick(targetPlan: PlanTier) {
  // Free → Free
  if (currentPlan === 'free' && targetPlan === 'free') return

  // Same paid plan
  if (currentPlan !== 'free' && targetPlan === currentPlan) return

  // Upgrade
  if (isUpgrade(targetPlan, currentPlan)) {
    startStripeCheckout(targetPlan)
    return
  }

  // Downgrade (paid → lower paid or free)
  router.push('/settings/billing')
}
```
