Implement a premium desktop/tablet redesign for the GigAid “Notify Clients” page WITHOUT breaking existing functionality.

CRITICAL RULES
1. Do NOT break current mobile behavior.
2. All redesigned UI should only apply to tablet and desktop.
3. Mobile view below 768px must remain exactly as it is today unless absolutely required for shared logic.
4. Do NOT change database schema.

6. Reuse existing services, campaign, and notification data where possible.
