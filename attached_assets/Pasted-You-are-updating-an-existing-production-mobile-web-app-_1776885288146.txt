# 🔵 PART 1 — FIX SIGN IN WITH APPLE FLOW

## GOAL:

Ensure the app does NOT request name or email after Apple authentication.

## CURRENT ISSUE:

The app prompts users to re-enter name/email after using Apple Sign In.

## REQUIRED FIX:

### 1. Update Apple Auth Handler

When Apple returns authentication data:

* Extract:

  * userId (sub)
  * email
  * fullName (ONLY on first login)

### 2. AUTO-CREATE USER

If user does not exist:

* Create user using:

  * apple_user_id = sub
  * email = email (can be relay email)
  * name = fullName (if available)

If fullName is missing:

* DO NOT prompt user
* Store name as null or “User”

### 3. REMOVE ALL POST-LOGIN FIELDS

Find and REMOVE any UI that:

* Asks for email after Apple login
* Asks for name after Apple login

🚫 These MUST NOT appear after Apple login.
