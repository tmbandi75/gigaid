## 1Ô∏è‚É£ Text-to-Plan AI
### Goal: Convert freeform text or voice messages into structured job entries.

### UI Component:
`<TextToPlanInput />`  
- Accepts text or voice input  
- Displays editable job suggestion

### API: `/api/ai/parseJobPlan`
- Input: `{ message: string | audioBlob, userId: string }`
- Output: `{ service, date, time, clientName }`

### Flow:
1. If audio: transcribe with Whisper
2. Parse transcription with GPT/Claude
3. Display editable job preview
4. Save to `jobs` table
