## 2️⃣ Enforce Job Limit (Free → Pro Capacity Gate)

### Create helper:

`/src/lib/canCreateJob.ts`

```ts
import { PLAN_LIMITS } from "./planLimits";
import { Plan } from "./plans";

export function canCreateJob({
  plan,
  currentJobCount
}: {
  plan: Plan;
  currentJobCount: number;
}) {
  const limit = PLAN_LIMITS[plan].maxJobs;
  return currentJobCount < limit;
}
```

### Use in job creation flow:

```ts
if (!canCreateJob({ plan: user.plan, currentJobCount })) {
  showBlockingMessage({
    title: "Job limit reached",
    message:
      "Free includes up to 10 jobs. Upgrade to Pro for unlimited jobs."
  });
  return;
}
```

⚠️ Must only block **new job creation**
❌ Do NOT block viewing, editing, completing jobs
