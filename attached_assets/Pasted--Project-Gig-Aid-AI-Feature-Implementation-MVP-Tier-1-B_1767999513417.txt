## üß† Feature 1: Text-to-Plan AI
**Goal:** Convert freeform text or voice note into a structured job (service, date, time, client)

### Frontend:
- Component: `<TextToPlanInput />`
- Modes: Text or Voice (toggle)
- On submit: Call `/api/ai/parseJob`

### Backend Endpoint: `/api/ai/parseJob`
- Input: `{ message: string }`
- Use OpenAI or Claude to extract:
  - service
  - date (natural language ‚Üí ISO)
  - time
  - client name (if included)
- Return: `JobDraft` object `{ service, date, time, clientName }`

### Bonus:
- Use Whisper to transcribe voice ‚Üí text before calling GPT
- Use `react-speech-recognition` for voice UI

## üìÖ Feature 2: Smart Scheduling Assistant
**Goal:** Suggest 3 optimal time slots based on:
- User availability (from Supabase table `availability`)
- Existing jobs (table `jobs`)
- (Optional) Travel time between jobs (via Google Maps API)

### Frontend:
- Component: `<SuggestedSlots jobDuration={minutes} location={latlng} />`
- On job creation, show suggested slots with "Accept" or "Pick manually"

### Backend:
- Endpoint: `/api/schedule/suggestSlots`
- Input: `{ userId, jobDuration, jobLocation }`
- Logic:
  1. Fetch availability blocks
  2. Fetch existing jobs
  3. Exclude overlaps
  4. (Optional) Add travel time buffers
  5. Return next 3 available time blocks
